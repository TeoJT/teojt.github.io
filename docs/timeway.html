Timeway is my main and biggest project. Initially designed to be a journal-keeper, it is a file manager where the files are shown in a 3D retro world, and directories are navigated through portals. Each folder holds a world, called "realms" in Timeway.



There are many features, including:

  • Running and jumping through realms, and entering portals representing folders.
  • Picking up and moving files- this moves your actual files on your computer.
  • Creating entries (these are similar to Microsoft OneNote).
  • Customising realms with skies, grass and tree textures as well as background music.
  • Building realms by shaping the ground and growing trees and objects.
  • A choice of 100 realm templates to choose from when you enter a new folder.
  • File indexer search.
  • Standard file stuff like renaming, moving to recycle bin, copying, creating shortcuts etc.
  • Caching and many performance-focused features in the background.




The project started off as a project way to create notes and journal entries in a fun interactive way, but integrated a concept where information could be interacted with in a 3D space, as if your computer was a world you could walk in. Timeway is built on top of the Processing framework.




In Timeway, I focus on performance, robustness and backward-compatibility. Timeway is optimised to run well on any device, by using caching and reducing graphical overheads. I also minimise any technical debt by fixing bugs as soon as I discover them. As I create many files in Timeway, backward-compatibility becomes a big priority; any file created in early versions must be loadable in the latest version.




As of now, the codebase sits at almost 30,000 lines of code (including whitespaces and comments), and there's still many planned things to add, such as:

  • File inventory and realm asset manager.
  • Image editor (useful for creating realm sky, grass, and tree textures).
  • Other file editors like an audio editor.
  • Interaction with more file types like .zip and video files.
  • More optimisations to make Timeway even faster.
  • Maybe even multiplayer so other users can visit your realms...

The long-term goal is to make Timeway a multi-purpose application with a 3D explorer at its center.

Once complete, extract the files from the zip folder and run "Timeway.exe" on Windows, or right-click and click "Run as application" on Linux.
On Windows, you may get a "Windows protected your PC" warning. Click "more info" and click "run anyway".
The reason for this warning is because the application is not signed. Signing it would require purchasing an expensive certificate, which, alas, I cannot afford, sadly.
