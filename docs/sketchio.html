Welcome to Sketchio, a work-in-progress, yet-to-be-released animation program where you can create animations with code.



Sketchio is the successor to my older project called Sketchiepad, which did a lot of heavy-lifting in animated Processing sketches such as loading images, caching large files, music playback, and moving objects. However, there was a lack of UI, missing features, and no real direction for the codebase. It was more of a library than an animation program.

Sketchio solves all this. It is a proper stand-alone application which loads animation scripts during runtime, and has many neat features to make animating faster. It can compile scripts during runtime, seek to any point in the animation using the time bar, render complete MP4 files, and provides many UIs which replaces Sketchiepad's code-based config files.


Under the hood, it uses the same custom engine which powers Timeway, as it is orientated towards file-management and media, perfect for Sketchio. All of that runs on top of the Processing framework, where its API is exposed alongside Sketchio's own API in projects.


Sketchio is still very much work-in-progress and not ready for download yet, but if you're still interested, you can explore the Github repository or check out this blog post which talks about Sketchio in more detail.
